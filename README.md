# Jeff's Jira Client

# Setup

* [Get a JIRA Token](https://confluence.atlassian.com/cloud/api-tokens-938839638.html)
* Download a binary for the [latest release](https://github.com/jeffutter/jjc/releases)
  * Put it in your path
  * Make it executable
* Set the following Environment Variables:
  * `JIRA_TOKEN`
  * `JIRA_USERNAME`
  * `JIRA_HOST`
* `jjc hel`
